Cómo usar el script

1. Crear el archivo: Copia el código anterior y pégalo en un archivo llamado monitor_recursos.sh.

2. Dar permisos de ejecución: Abre una terminal y navega hasta el directorio donde guardaste el archivo. Luego, ejecuta el siguiente comando para dar permisos de ejecución al script:

chmod +x monitor_recursos.sh

3. Ejecutar el script: Inicia el script con el siguiente comando:

./monitor_recursos.sh

Detalles sobre cómo funciona el script

- Menú de recursos: Al ejecutarse, el script muestra el uso de CPU, RAM y disco duro de manera visual.

- Gráficos de uso: Utiliza caracteres ASCII para representar gráficamente el uso de recursos.

- Actualización: El script permite actualizar la información en tiempo real hasta que el usuario decida salir.

Notas adicionales

- Este script requiere que se ejecute en un entorno de terminal compatible con ANSI para que los colores se muestren correctamente.

- Asegúrate de tener los permisos necesarios para ejecutar los comandos utilizados en el script.
